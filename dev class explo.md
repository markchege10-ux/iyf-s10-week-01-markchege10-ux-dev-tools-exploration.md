## Website 1 (https://example.com)
**What HTML tags are used on the page?**
* Html: root element of the page
* Head: contains meta-info/machine readable code
* Body: the container for all visible content on the page

   * h1: main heading
   * p>: paragraphs of text
   * a>: anchor tags for links
               
* iframe: used to embed another document within the current one
* Nt-app: a custom element (likely specific to chrome "new tab page)
* Div: a generic container used for styling or grouping
* svg: used fo displaying vector graphics
* Script: used to embed or reference executable client-side reports
* Link: defines the relationship between the document and external resources
* !DOCTYPE.html: document type declaration

    * Whem you hover over links on the website, you'll notice several changes. these are controlled by css, you will see
